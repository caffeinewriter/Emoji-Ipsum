Emoji Ipsum

This is really just a silly, mostly useless experiment that will probably not work properly for all browsers, but was a fun hack nonetheless. It's just a simple generator for lots of random emoji. Desktop Chrome will likely not work with this, at least not at the time of creation. You can get it to work by using the lovely Chromoji addon. It should work in Firefox and Safari at least. As for the pervasive Internet Explorer, get a modern browser. There's a lot of work to be done on this. Please check out the issues page if you're interested in contributing.
